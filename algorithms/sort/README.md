##归并排序
概念：在一个数组中，取中间下标把数组划分为两个区域，把把已经划分的区域两两划分，直到单个无法划分为止。然后进行比较后有序合并。
归并排序也是递归的一种运用
稳定性：稳定
空间复杂度：非原地排序算法，需要额外的存储空间
时间复杂度：
1. 最好情况：O(nlogn)
2. 最坏情况：O(nlogn)
3. 平均情况：O(nlogn)

##快速排序
概念：在一组数据中，定好一个分区点，分区点左边的数据都小于分区点，分区点右边的数据都大于分区点。然后分区中重复定好分区点，排好数据的操作。
快排也是递归的一种运用。
稳定性：不稳定
空间复杂度：原地排序算法
时间复杂度：
1. 最好情况：O(nlogn)
2. 最坏情况：O(n^2)
3. 平均情况：O(nlogn)
